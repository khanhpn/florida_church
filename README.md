# How to contribute
- Clone source code from develop: `git clone git@github.com:khanhpn/florida_church.git`
- Run `bundle install`
- Run `rails db:create`
- Run `rails db:migrate`
- Run seed to generate sample admin account: `rails db:seed`
- Run `yarn install` or `npm install`

# Technology
- Ruby
  - 3.0.0
- Rails
  - Rails 6.1.4.1
- Postgresql
- Tinymce
- Cloudinary
- AdminLTE
- Node 
  - v16.13.0
